# API Documentation

## Public

### `GET /health`

Returns service health.

## Auth

### `POST /auth/login`

Body:

```json
{
  "email": "admin@roiscraft.ai",
  "password": "ChangeMeNow!2026"
}
```

Returns a JWT bearer token for admin routes.

## Telegram

### `POST /telegram/webhook`

Receives Telegram updates. The request must include `X-Telegram-Bot-Api-Secret-Token` matching `TELEGRAM_WEBHOOK_SECRET`.

Handled updates:

- text messages
- inline callback queries
- handoff requests
- VIP routing requests

## Admin

All admin routes require:

```http
Authorization: Bearer <token>
```

### `GET /admin/metrics`

Returns dashboard metrics generated by the Supabase `dashboard_metrics()` function.

### `GET /admin/users`

Optional query parameters:

- `stage`
- `temperature`

Returns Telegram CRM profiles with qualification stage, lead temperature, score, and follow-up status.

### `GET /admin/users/export.csv`

Exports CRM users as CSV.

### `GET /admin/users/{user_id}/messages`

Returns the stored Telegram conversation history for admin review.

### `POST /admin/users/{user_id}/notes`

Adds an admin note to the audit log.

### `GET /admin/tasks`

Query:

- `status=open`

Returns action-required tasks.

### `PATCH /admin/tasks/{task_id}`

Updates task status, assignment, or notes.

### `POST /admin/documents`

Multipart form:

- `file`: PDF, DOCX, TXT, MD
- `portfolio`: `ROISCRAFT`, `Aurum Foundation`, `Bytnet`, or future portfolio name

Extracts text, chunks content, generates embeddings, and stores vectors in Supabase pgvector.

### `POST /admin/community-groups`

Upserts configurable Telegram links.

```json
{
  "key": "vip_group",
  "portfolio": "ROISCRAFT",
  "label": "ROISCRAFT VIP Investor Onboarding",
  "invite_url": "https://t.me/+...",
  "min_stage": "HIGH_INTENT",
  "is_vip": true,
  "is_active": true
}
```

### `GET /admin/community-groups`

Lists configured Telegram groups, VIP links, webinar links, and resource destinations.

### `POST /admin/broadcasts`

Creates a segmented campaign record for scheduled automation.

### `POST /admin/broadcasts/{broadcast_id}/dispatch`

Dispatches a broadcast to the segment immediately and records engagement events.

### `POST /admin/reports/daily`

Generates the daily intelligence report and sends it to configured Telegram admin targets.

### `POST /admin/telegram/webhook`

Registers the Telegram webhook using `APP_BASE_URL` and `TELEGRAM_WEBHOOK_SECRET`.
